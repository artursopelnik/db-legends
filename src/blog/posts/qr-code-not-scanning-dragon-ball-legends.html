You point the in-game scanner at a QR code and… nothing happens. Scanning problems in Dragon Ball Legends almost always come down to a handful of causes – here are the fixes, ordered by how often they solve the problem.

1. Regenerate the QR code (most common fix)

Dragon Ball Legends only accepts QR codes with a fresh timestamp. A QR code generated a few hours ago – or a screenshot someone sent you yesterday – will simply be ignored by the scanner. Open the QR generator, tap Regenerate, and scan immediately. This fixes the majority of "scanner does nothing" cases.

2. Check that a scan event is actually running

The Scan QR code option in Menu → Friends is only active during scan campaigns (typically anniversary and Legends Festival periods). Outside those windows the scanner isn't available at all – no QR code will work, no matter how fresh it is.

3. Verify the friend code itself

A QR code generated from a mistyped friend code scans fine but points to the wrong (or no) player. Friend codes are 8–10 letters and numbers – double-check for classic mix-ups like 0 vs o and 1 vs l. Ask your friend to copy the code directly from their game (Menu → Friends) instead of typing it from memory.

4. Fix the screen, not the code

  • Brightness up on the device displaying the QR code.
  • Avoid glare – tilt the screen away from lamps and windows.
  • Make it bigger – tap the QR code in the generator to show it fullscreen.
  • Disable night mode / blue-light filters, which reduce contrast.

5. You already scanned this player

Each unique friend can only give you one Dragon Ball per event period. If the scanner reads the code but nothing is awarded, you've probably scanned that player already – you need codes from different friends. Community friend-code threads are a good source; see how to find and share friend codes.

6. Camera and permission issues

If the scanner screen stays black, the game may be missing camera permission. Check your phone's app settings, grant camera access to Dragon Ball Legends, and restart the game. Cleaning the camera lens helps more often than you'd think.

7. Restart the game

The classic. If the event is live, the code is fresh and the camera works but scanning still fails, close Dragon Ball Legends completely and reopen it. Server hiccups during busy anniversary days can also cause temporary failures – waiting a few minutes and trying again is sometimes all it takes.
